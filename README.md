# Low level Calculator

Super smart and up-to-date calculator for regular use. Soft and pleasant purple glow pleasantly pleases the eye.

### For run
```bash
> git clone https://github.com/vicimpa/easy-calc
> npm install
> npm run dev
```

### For build
```bash
> git clone https://github.com/vicimpa/easy-calc
> npm install
> npm run build
```
